# Drox IDE 1.3.0

## Nouveautés

- Première distribution packagée Windows (installeur user).
- Moteur Drox embarqué (`resources/drox/win32-x64/drox.exe`).
- Chat Drox et intégration Ollama.
- Branding Drox IDE (icônes, installeur noir/vert).

## Prérequis

- Windows 10 ou plus récent (64 bits)
- [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)
- [Ollama](https://ollama.com/) pour le chat local (recommandé)

## Installation

Télécharger `Drox-IDE-Setup-1.3.0-win32-x64.exe` depuis les [Releases](https://github.com/DroxKiwi/Drox---IDE---releases/releases) et suivre l’assistant.
